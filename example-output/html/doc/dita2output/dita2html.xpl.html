p:declare-step

Make XHTML-documentation from DITA files.

Description

This step receives a list of DITA files, loads them, transforms them into XHTML using an XSLT stylesheet (dita2html.xsl), and stores them to the output directory. The list of DITA files is also used to produce a table of contents of all files so that the documentation is easier to navigate.
