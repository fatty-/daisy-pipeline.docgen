lib/recursive-directory-list.xpl

p:declare-step

Performs p:directory-list recursively.

Description

The cx:recursive-directory-list step lists the contents of the specified directory on the filesystem. Unlike the standard p:directory-list step, cx:recursive-directory-list expands all subdirectories recursively.
