document.xpl

p:declare-step

Main entry point for the documentation module.

Description

Use this script to document a module. The script will first instruct makedita.xpl to generate documentation for the module, which will be in the DITA format. Then the script instructs dita2html.xpl to use the resulting DITA files to generate documentation in HTML. The documentation will be generated in the 'doc'-directory in the module.

d:makeDita - dita

Make DITA files.

Description

Iterates through all files in the module and makes DITA documents of them.

Options

namerequiredconnectiondescription
srcPath
tempPath
ditaPath
selfPath

Source Code

<d:makeDita name="dita" xmlns:p="http://www.w3.org/ns/xproc" xmlns:d="http://www.example.org/documenter">
        <p:with-option name="srcPath" select="$srcPath"/>
        <p:with-option name="tempPath" select="concat($docPath,'temp')"/>
        <p:with-option name="ditaPath" select="concat($docPath,'dita')"/>
        <p:with-option name="selfPath" select="$selfPath"/>
    </d:makeDita>

d:dita2html

Make XHTML5-documentation.

Inputs

nameprimarysequenceconnectiondescription
source true result@dita List of files.

Options

namerequiredconnectiondescription
title
ditaPath
tempPath
htmlPath
selfPath

Source Code

<d:dita2html xmlns:p="http://www.w3.org/ns/xproc" xmlns:d="http://www.example.org/documenter">
        <p:input port="source">
            <p:pipe port="result" step="dita"/>
        </p:input>
        <p:with-option name="title" select="replace($srcPath,'^.*/','')"/>
        <p:with-option name="ditaPath" select="concat($docPath,'dita')"/>
        <p:with-option name="tempPath" select="concat($docPath,'temp')"/>
        <p:with-option name="htmlPath" select="concat($docPath,'html')"/>
        <p:with-option name="selfPath" select="$selfPath"/>
    </d:dita2html>
